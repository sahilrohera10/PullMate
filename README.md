# PullMate 🚀  
**Your companion for streamlined PR reviews**  

PullMate automates **pull request (PR) reviews**, providing AI-driven insights, reviewer suggestions, and real-time comments on PRs and commits.

## ✨ Features  
- **GitHub OAuth Login** – Securely connect your GitHub account.  
- **Repository Selection** – Fetch and manage your repositories from the platform.  
- **Automated PR Reviews** – Get AI-powered suggestions and improvements on every new PR and commit.  
- **Seamless Workflow Deployment** – Deploy review automation in just a few clicks.

## 🛠️ Tech Stack  
- **Frontend:** Next.js  
- **Backend:** Node.js & TypeScript  
- **Database:** PostgreSQL  
- **AI Analysis:** LLMs & APIs  
- **Authentication:** GitHub OAuth
